# Tomorrow People

A creative community exploring technology, art, and better ways to live and build together.

## Project Structure

```
tmw.ppl/
├── css/
│   ├── variables.css    # CSS custom properties and theme variables
│   ├── base.css         # Base styles and resets
│   ├── components.css   # Reusable UI components
│   ├── layout.css       # Layout and navigation styles
│   ├── typography.css   # Typography and text styles
│   └── pages.css        # Page-specific styles
├── js/
│   ├── main.js          # Core functionality (smooth scroll, mobile menu)
│   └── events.js        # Events page functionality
├── assets/              # Images, icons, and other assets
├── index.html           # Home page
├── events.html          # Events page
└── README.md           # This file
```

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile
- **Dark/Light Theme**: Automatically adapts to user's system preference
- **Modern CSS**: Uses CSS Grid, Flexbox, and custom properties
- **Accessible**: Semantic HTML with ARIA labels and keyboard navigation
- **Performance**: Optimized fonts and minimal JavaScript

## Getting Started

1. Clone or download the project
2. Open `index.html` in a web browser
3. No build process required - it's vanilla HTML, CSS, and JavaScript

## Customization

- **Colors**: Edit `css/variables.css` to change the color scheme
- **Typography**: Modify `css/typography.css` for font changes
- **Layout**: Adjust `css/layout.css` for spacing and structure
- **Content**: Update the HTML files with your content
